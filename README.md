# be_happy
TD — Treatment of Depression — это десктопное приложение с мотивирующими цитатами и ИИ-чатом поддержки, помогающее людям справляться с депрессией, тревогой и потерей смысла. Проект ориентирован на людей всех возрастов и предлагает доступную эмоциональную помощь без медикаментов и психологов.
